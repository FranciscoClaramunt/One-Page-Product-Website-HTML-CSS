# One-Page-Product-Website-HTML-CSS
Contains an academic project developed for the Web Interface Design course at Universidad ORT Uruguay.
The project consists of a one-page product website created using semantic HTML5 and external CSS, based on a wireframe provided by the course. It focuses on layout structure, visual hierarchy, typography, internal anchor navigation and overall interface coherence.

## Features
- One-page layout with internal anchor navigation
- Semantic HTML5 structure
- External CSS with variables and reusable classes
- Custom typography using Google Fonts
- Product sections, pricing table and form
- Cohesive visual identity and layout hierarchy

## Tools Used
- HTML5
- CSS3
- Google Fonts
